# 🏦 Bank Management System (Python)

## Overview

This is a console-based Bank Management System developed using Python. The project demonstrates Object-Oriented Programming (OOP), File Handling, and modular programming concepts.

The system allows users to perform common banking operations while securely storing account information in files.

---

## Features

- Create New Account
- Login using Account Number and PIN
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Transfer Money
- View Account Details
- Delete Account
- Persistent Data Storage using File Handling
- Menu-driven Console Application

---

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
